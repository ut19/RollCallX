About Cloud Attendance

Cloud Attendance is a premium, secure, and modern attendance tracking application. Designed to simplify classroom management, it uses advanced verification techniques to keep data secure and reliable.

🔒

Secure Password Hashing

All user passwords undergo industrial-grade, secure hashing (scrypt/PBKDF2) before storage, eliminating any exposure of raw text credentials.

💾

SQLite Storage Engine

Migrated from transient JSON configurations to SQLite3 relational persistence. Ensures concurrent write safety and data permanence across server restarts.

📍

Geolocation Verification

Verifies physical classroom attendance by validating real-time GPS coordinates of student submissions directly in the browser.

📄

Proof of Work Uploads

Enables file attachments (such as PDF, JPG, PNG daily work files) alongside geolocation coordinates for comprehensive evidence of active attendance.
